\section{Observations}

From the experimental results, the following key observations were made:

\subsection{Fixed Ciphertext Size}

\textbf{Observation:} The encrypted size remains nearly constant (approximately 362 bytes) for all fields, regardless of the original plaintext size \cite{cryptoeprint:2023/404}.

\begin{itemize}
    \item Protects against information leakage via data size.
    \item Ensures uniform security across records.
\end{itemize}

\subsection{Processing Efficiency}

\textbf{Observation:} The system achieved a high throughput of approximately 384,000 encryptions per second \cite{zhao2025vlwe}.

\begin{itemize}
    \item The Encryption Phase dominates the overall processing time.
    \item Setup and Encoding overheads are minimal.
\end{itemize}

\subsection{Storage Overhead}

\textbf{Observation:} Significant storage expansion (~62.12x) was observed.

\begin{itemize}
    \item Plaintext Database: 34 MB; Encrypted Database: 2.1 GB.
    \item Impacts storage, bandwidth, and memory requirements.
\end{itemize}

\subsection{Distribution Analysis}

\textbf{Observation:} Two distinct expansion ratio groups were found:

\begin{itemize}
    \item Binary fields (such as \texttt{isFraud}) exhibited maximum expansion (~362x).
    \item Textual fields (such as \texttt{nameDest}) showed moderate expansion (~35x--50x).
\end{itemize}

\subsection{Security Considerations}

\textbf{Observation:} The encryption preserves 128-bit security and prevents structural information leakage \cite{cryptoeprint:2024/2041}.

\section{Recommendations}

Based on the observations, the following recommendations are made:

\begin{itemize}
    \item Prioritize encryption for sensitive fields rather than encrypting the entire dataset.
    \item Investigate post-encryption compression strategies to reduce storage overhead.
    \item Explore ciphertext packing techniques to combine multiple small values into a single ciphertext.
    \item Optimize encryption parameters based on application-specific security and performance requirements.
\end{itemize}
